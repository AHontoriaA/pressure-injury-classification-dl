# pressure-injury-classification-dl
Deep learning framework for pressure injury classification with clinical validation

# Pressure Injury Classification using Deep Learning

This repository contains the code developed for a Master's Thesis focused on the classification of pressure injuries using deep learning and clinical image data.

Note: Notebooks are provided without outputs to reduce file size.

## Dataset
The original dataset is not publicly available due to clinical confidentiality.

A small set of example images is provided for demonstration purposes.
